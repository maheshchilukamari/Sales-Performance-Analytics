# Screenshots

After building the Power BI dashboard, export screenshots and place them in this folder. These images make the GitHub project easier for recruiters and hiring managers to review without opening Power BI Desktop.

Recommended screenshot files:

- `dashboard_overview.png`
- `monthly_sales_trend.png`
- `sales_by_region.png`
- `profit_by_category.png`
- `top_10_products.png`
- `customer_segment_analysis.png`

## Screenshot Tips

- Use clear, high-resolution screenshots.
- Capture the full dashboard overview first.
- Include individual screenshots for the most important visuals.
- Keep file names lowercase and descriptive.
- Add screenshots to the README after the final Power BI dashboard is created.
